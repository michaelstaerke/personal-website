Meine Server / Anwendungen



Ich habe für praktische Lern- und Testzwecke mein eigenes Heimnetzwerk durch verschiedene IT-Systeme erweitert.


Von diverser Hardware von z.B. Lan-Kabeln (cat5 - cat8), Switches (D-Link unmanaged, Netgear webmanaged), über zu eigenen Servern (Ubuntu-Server-Laptop, Raspberry Pi Server, virtualisierten Servern auf meinem Administrator-Desktop-PC). Für die Zukunft sind weitere Geräte geplant wie z.B. ein NAS-Server (Backups, Filesharing), verschiedene Gadgets für das Smart Home (Monitoring Screens z.B. für den Raspberry Pi Cluster, Strommesser, etc.)



Um genauer auf die Server einzugehen, beschreibe ich mal kurz die Anwendungen.



Auf dem Administrator-Desktop-PC (mit Windows 10 Pro Edition) befinden sich IT-Lösungen und verschiedene Testumgebungen wie z.B.

- Windows Subsystem für Linux (Ubuntu, Debian, Kali)

- Microsoft Azure Cloud Testversion

- virtualisierte Server mit Virtualbox (Backup Server für das gesamte Heimnetzwerk, von Sicherungen für das Android Handy, über zu Sicherungen für verschiedene PCs mit ihren verschiedenen Betriebssystemen wie Windows und Linux; Mailserver, Druckserver, Datenbankserver, Proxyserver)

- weitere Betriebssysteme zu Lern- und Testzwecken in der Virtualbox und WMware Workstation (Debian, Ubuntu, Fedora, CentOS, Arch Linux)



Auf dem Ubuntu-Server-Laptop (älterer Asus Laptop auf dem das Ubuntu-Server-Betriebssystem ohne visuelle Oberfläche, also nur die Konsole läuft) befindet sich:

- OpenVPN Server




Und zum Schluss der Raspberry Pi 3 Model B (eigens für dieses kleine Wunderwerk habe ich ein Raspberry Pi Cluster Gehäuser gekauft, sodass ich zukünftig mehrere Raspberry Pis im Dauerbetrieb mit verschiedenen Anwendungen, laufen lassen kann = Raspberry Pi Cluster) darauf befindet sich bisher:

- Apache Webserver mit dem beliebten LAMP Stack zum Hosten dieser Website (www.michael-staerke.de) im Dauerbetrieb

- Pi-hole zum Unterbinden von Trackung und Werbung für das gesamte Heimnetzwerk
