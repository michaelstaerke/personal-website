Über mich

Hallo, ich heiße Michael Stärke und mein Spitzname ist Mike. Ich bin 26 Jahre alt und wohne derzeitig im schönen Konstanz am Bodensee. Ursprünglich komme ich aus dem Landkreis Lörrach, welches benachbart ist zu Frankreich und zur Schweiz, also das Dreiländereck Südwest.

Privat bin ich in erster Linie sehr viel am Surfen im Web, täglich erkunde ich die neusten IT-News oder allgemeine IT-Artikel auf Youtube oder in Online Magazinen wie Entwickler.de, Heise, Golem, Netzpolitik, T3N oder in englischsprachigen Magazinen. Weil ich schon seit Kindestagen viel am PC saß, verschiedenste Spiele spielte oder viel Zeit auf Youtube verbrachte, tat mir damals mein erstes richtiges Hobby nämlich Basketball, in vielerlei Hinsicht richtig gut. Projekte Zusammenfassung

Personal website build from scratch

Okay okay, Experten werden Bootstrap 5 sehen und direkt denken, ne ne das ist nicht "from scratch", ja diesem Blickwinkel gebe ich auch Recht. Ich wollte hiermit den Industriestandard für CSS Frameworks und zwar Bootstrap üben und auch in 2, 3 oder 5 Jahren nicht gezwungen sein, alles von neu zu Programmieren.

Diese Website entstand nach meinem Udemy Online Kurs "Der komplette Webentwickler 2.0" dort bekam ich erstens wichtiges Wissen über die Webentwicklung und zweitens verschiedene Werkzeuge zur Erstellung von Websiten und dazugehörigen Anwendungen wie Datenbanken, an die Hand.

Folgende Webtechnologien verwende ich:
HTML, CSS, das CSS-Framework Bootstrap 5, JavaScript, PHP, MySQL, das FTP-Programm FileZilla, die Entwicklungsumgebung Visual Studio Code, das Versionskontrollsystem Git und Github

coming soon ~ Website Besucher Tracking bzw. Monitoring System

Als ich die anderen Wordpress Websiten administrierte, fiel mir auf, dass über die angezeigten Statistiken der IT-Security-Plugins viel "Hintergrund" Aktivität zu sehen ist. Verschiedene Website Besucher aus verschiedenen Ländern, manche probierten Schwachstellen-Angriffe auf den Websiten aus. Nicht jede IP Adresse muss zwangsweise ein "Hacker" sein, es könnten auch Bots bzw. Skripts von solchen Hackern sein, die alles und jeden im Netz nach Schwachstellen angreifen.

Diese Software soll das gleiche Thema behandeln, quasi ein kleines Monitoring System der Besucher dieser Website. Website-Besucher Daten werden über PHP in einer MySQL Datenbank auf meinem LAMP-Webserver gespeichert und dann werden aus diesen Daten interessante Statistiken generiert. Wie z.B. wie viele Besucher / Bots hat meine Website täglich oder aus welchen Ländern kommen diese Besucher (IP-Zuordnungen)? Wie lange verweilen die Besucher und wo probieren Hacker-Skripts Schwachstellen auf meiner Website aus?
